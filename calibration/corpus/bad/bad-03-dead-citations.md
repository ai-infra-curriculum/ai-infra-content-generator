# Further Reading: Agent Architectures

This lesson collects the canonical references for multi-agent system design.
Work through them in order before attempting the capstone.

## Primary sources

- **The ReAct paper** — read the full write-up at
  [research.example-ai-lab.org/papers/react-2022-final.pdf](https://research.example-ai-lab.org/papers/react-2022-final.pdf).
- **Orchestrator-Worker patterns** — the definitive guide lives at
  [docs.agentframework.io/v2/patterns/orchestrator](https://docs.agentframework.io/v2/patterns/orchestrator)
  (note: the v2 docs replaced the old v1 site, which is now offline).
- **Tool-calling spec** — see the official specification at
  [toolcalling-standard.org/spec/latest](https://toolcalling-standard.org/spec/latest).
