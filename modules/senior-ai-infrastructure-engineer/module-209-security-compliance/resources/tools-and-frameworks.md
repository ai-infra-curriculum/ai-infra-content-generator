# Module 209: Tools and Frameworks

## Security Scanning

**Trivy**
- Container vulnerability scanning
- Kubernetes config scanning
- License scanning

**Grype**
- Vulnerability scanner for container images
- SBOM analysis

**kube-bench**
- CIS Kubernetes Benchmark checker
- Automated security assessment

## Policy Enforcement

**OPA Gatekeeper**
- Policy enforcement for Kubernetes
- Rego policy language
- Admission control

**Kyverno**
- Kubernetes-native policy management
- YAML-based policies
- Validation, mutation, generation

## Service Mesh

**Istio**
- mTLS for service-to-service
- Authorization policies
- Traffic management

**Linkerd**
- Lightweight service mesh
- Automatic mTLS
- Lower resource overhead

## Secrets Management

**HashiCorp Vault**
- Secret storage and rotation
- Dynamic secrets
- Encryption as a service

**External Secrets Operator**
- Sync secrets from external sources
- Supports AWS, GCP, Azure KMS
- Automatic rotation

**TensorFlow Privacy**
- DP-SGD implementation
- Privacy accounting
- Analysis tools

**PySyft**
- Federated learning
- Secure multi-party computation
- Differential privacy

**Opacus** (PyTorch)
- Differential privacy library
- DP-SGD for PyTorch
- Privacy analysis
