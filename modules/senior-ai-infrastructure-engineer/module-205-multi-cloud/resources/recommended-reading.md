# Module 205: Multi-Cloud Architecture - Recommended Reading

## Books

### Essential Reading

1. **"Cloud Native Infrastructure"** by Justin Garrison and Kris Nova
   - Publisher: O'Reilly Media
   - Focus: Building cloud-native systems across providers
   - Why: Foundational concepts for multi-cloud architecture

2. **"Cloud FinOps"** by J.R. Storment and Mike Fuller
   - Publisher: O'Reilly Media
   - Focus: Cost optimization and financial operations
   - Why: Critical for managing multi-cloud costs

3. **"Site Reliability Engineering"** by Google
   - Publisher: O'Reilly Media
   - Focus: Building reliable distributed systems
   - Why: DR and reliability patterns applicable to multi-cloud

### Advanced Reading

4. **"Designing Distributed Systems"** by Brendan Burns
   - Publisher: O'Reilly Media
   - Focus: Patterns for building distributed systems
   - Why: Essential patterns for multi-cloud applications

5. **"Network Programmability and Automation"** by Jason Edelman et al.
   - Publisher: O'Reilly Media
   - Focus: Automating network configuration
   - Why: Multi-cloud networking automation
