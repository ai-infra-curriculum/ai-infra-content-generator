# Recommended Reading for Module 201

## Books

### Essential Reading

1. **"Programming Kubernetes" by Michael Hausenblas & Stefan Schimanski**
   - Publisher: O'Reilly Media
   - ISBN: 978-1492047100
   - Topics: Operator development, Custom Resources, Controller runtime
   - Why: Best resource for understanding Kubernetes internals and building operators
   - Level: Intermediate to Advanced

2. **"Kubernetes Patterns" by Bilgin Ibryam & Roland Hu√ü**
   - Publisher: O'Reilly Media
   - ISBN: 978-1492050285
   - Topics: Design patterns for cloud-native applications
   - Why: Essential patterns for production Kubernetes deployments
   - Level: Intermediate

3. **"Production Kubernetes" by Josh Rosso, Rich Lander, Alex Brand, John Harris**
   - Publisher: O'Reilly Media
   - ISBN: 978-1492092308
   - Topics: Running Kubernetes in production, best practices, security
   - Why: Comprehensive guide to production operations
   - Level: Advanced

### Supplementary Reading

4. **"Cloud Native DevOps with Kubernetes" by John Arundel & Justin Domingus**
   - Publisher: O'Reilly Media
   - ISBN: 978-1492040767
   - Topics: Cloud-native applications, CI/CD, monitoring
   - Why: Practical guide to Kubernetes ecosystem
   - Level: Intermediate

5. **"Kubernetes Security and Observability" by Brendan Creane & Amit Gupta**
   - Publisher: O'Reilly Media
   - ISBN: 978-1098107109
   - Topics: Security hardening, network policies, observability
   - Why: Deep dive into security best practices
   - Level: Advanced

6. **"Istio: Up and Running" by Lee Calcote & Zack Butcher**
   - Publisher: O'Reilly Media
   - ISBN: 978-1492043782
   - Topics: Service mesh, traffic management, security
   - Why: Comprehensive Istio guide
   - Level: Intermediate

7. **"Site Reliability Engineering" by Betsy Beyer et al.**
   - Publisher: O'Reilly Media
   - ISBN: 978-1491929124
   - Topics: SRE principles, incident response, monitoring
   - Why: Foundation for operating production systems
   - Level: All levels

## Reading Plan

### Week 1-2: Operators and CRDs
- Kubebuilder Book (chapters 1-5)
- "Programming Kubernetes" (chapters 1-4)
- Official Operator SDK documentation

### Week 3-4: Advanced Scheduling and Storage
- "Kubernetes Patterns" (relevant chapters)
- Kubernetes documentation on scheduling
- CSI specification and documentation

### Week 5-6: Networking and Service Mesh
- "Istio: Up and Running" (all chapters)
- Istio documentation and tutorials
- Cilium eBPF guides

### Week 7-8: Security and Multi-Cluster
- "Kubernetes Security and Observability"
- Pod Security Standards documentation
- KubeFed documentation

### Week 9-10: Production Operations
- "Production Kubernetes" (all chapters)
- "Site Reliability Engineering" (selected chapters)
- Case studies from KubeCon talks
