# Exercises — MOD-557 Governance & Compliance for ML Pipelines

1. **Policy-as-Code Implementation** — Integrate existing rule libraries (OPA, custom validators) into CI/CD and orchestration workflows.
2. **Evidence Automation** — Build scripts that collect metrics, experiment metadata, and validation results into an audit pack.
3. **Governance Review Simulation** — Conduct a mock governance board session documenting approvals, waivers, and follow-up actions.
