# Exercises — MOD-556 Automation & Orchestration at Scale

1. **Reusable Component Lab** — Create parameterized DAG components that can be imported across pipelines (training, validation, deployment).
2. **Policy Gate Integration** — Add security and governance checks (OPA, custom validators) into orchestration flows with automated rollbacks.
3. **Chaos & Resilience Testing** — Execute chaos experiments on pipelines and document mitigation strategies for incident response.
